## Installation

* From the command line run: `npm init -y`
*  `npm i express sequelize mysql2 dotenv`
*  Create a .env file in root folder
*  Enter MySQL login credentials in the following format:
*  `DB_NAME='ecommerce_db'`
*  `DB_USER='<username>'`
*  `DB_PW='<password>'`
